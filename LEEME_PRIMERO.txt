🚀 PRÓXIMOS PASOS (IMPORTANTE)

1️⃣ INSTALAR DEPENDENCIAS
   En terminal, en la carpeta jijasbet:
   
   npm install
   cd client && npm install && cd ..
   
   ⏱️ Esto toma 2-5 minutos

2️⃣ CONFIGURAR AMBIENTE
   
   a) Abre .env.example con un editor
   b) Cambia el nombre a .env
   c) Llena estas variables:
      - MONGODB_URI: Tu URL de MongoDB Atlas
      - JWT_SECRET: Una clave aleatoria
   
   💡 Tutorial en INSTALACION.md

3️⃣ CREAR USUARIOS DE DEMO
   
   node scripts/crearAdmin.js
   
   Esto crea 2 usuarios:
   - admin@test.com / admin123 (para verificar pagos)
   - usuario@test.com / 123456 (usuario regular)

4️⃣ INICIAR LA APLICACIÓN
   
   npm run dev
   
   Verás algo como:
   ✅ MongoDB conectado
   🚀 Servidor ejecutándose en puerto 5000

5️⃣ ABRIR EN NAVEGADOR
   
   http://localhost:3000
   
   ¡Ya está funcionando! 🎉

🌐 DEPLOYMENT (VERCEL)

Cuando quieras pasar a PRODUCCIÓN:

1. Sube código a GitHub:
   git push origin main

2. Ve a Vercel.com
   - Conecta tu GitHub
   - Importa el repositorio

3. Configura variables:
   MONGODB_URI = (tu URL de produción)
   JWT_SECRET = (clave segura)

4. Haz clic en Deploy
   ¡Listo! Tendrás URL como: https://jijasbet.vercel.app

⚠️ TROUBLESHOOTING

Problema: "Cannot find module 'express'"
Solución: npm install

Problema: "Port 5000 in use"
Solución: npx kill-port 5000

Problema: "MongoDB connection refused"
Solución: Verificar MongoDB URI en .env

Problema: "Admin panel no aparece"
Solución: node scripts/crearAdmin.js

Problema: "React page blank"
Solución: Abre DevTools (F12) y revisa console tab
